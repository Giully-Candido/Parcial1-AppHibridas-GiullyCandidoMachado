Documentación de la API REST – Excusas Creativas

Introducción

¿Llegaste tarde? ¿Te olvidaste de entregar algo? ¿Querés zafar de una reunión sin quedar mal? Tranquilo, tenemos la solución. Excusas Creativas es una API desarrollada con Node.js, Express y MongoDB que permite generar y gestionar excusas originales, divertidas y con distintos niveles de credibilidad para ayudarte a salir de cualquier situación incómoda.

Endpoints de Excusas

  • GET /api/excusas: Obtiene el listado completo de excusas.
  • GET /api/excusas/:id: Obtiene una excusa específica por su ID.
  • POST /api/excusas: Crea una nueva excusa.
  • PUT /api/excusas/:id: Actualiza una excusa existente.
  • DELETE /api/excusas/:id: Elimina una excusa.
  • GET /api/excusas?credibilidad=alta: Filtra por nivel de credibilidad (alta, media, baja)
  • GET /api/excusas?contexto=trabajo: Filtra por contexto (trabajo, universidad, familia, amigos, pareja)
  • GET /api/excusas?texto=abuela: Búsqueda por contenido del texto.

Endpoints de Contextos

  • GET /api/contextos: Obtiene el listado completo de contextos disponibles.
  • GET /api/contextos/:id: Obtiene un contexto específico por su ID.
  • POST /api/contextos: Crea un nuevo contexto.
  • DELETE /api/contextos/:id: Elimina un contexto existente por ID.
